Этот эксперимент - измерение влияния перемешивания вершин в слоях
и способа распределения вершин по потокам в слое. 

--inside_layer_schedule контролирует способ распределения вершин слоя по потокам
--shuffle_layers контролрует, перемешиваем ли мы вершины случайным образом перед распределениям

Результаты даны для
  - однопоточного жадного расписания
  - 4-х двухпоточных послойных расписаний - соотв опциям выше
  - 4-ч четырехпоточных послойных расписаний - соотв опциям выше

Надо бы еще запустить однопоточое послойное расписание

Результат эксперимента лежит в log.json
Соотвествтие с параметрами запуска можно восстановить по test_hardcode.bash
